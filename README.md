# Projeto Spotify - TCC MBA em Ciência de Dados

Este projeto foi desenvolvido como Trabalho de Conclusão de Curso do MBA em Ciência de Dados.  
O objetivo é analisar músicas do Spotify e identificar padrões relacionados à popularidade, gêneros e características sonoras.

## 📊 Dataset
- Fonte: Spotify Tracks Dataset
- Contém informações como: nome da faixa, artista, popularidade, duração, energia, dança, acústica, entre outros atributos.

## 🛠️ Tecnologias utilizadas 
- Python
- Pandas
- Matplotlib / Seaborn
- Scikit-learn

## 🔎 Principais análises
- Distribuição da popularidade das músicas
- Correlação entre atributos sonoros
- Identificação de padrões por gênero
- Modelos de machine learning para prever popularidade

## 🚀 Como executar
1. Clone ou baixe este repositório.
2. Abra o notebook `spotify_tcc_projeto.ipynb` no Jupyter Notebook ou JupyterLab.
3. Certifique-se de instalar as bibliotecas listadas no notebook.
4. Execute as células para reproduzir as análises.

## 📈 Resultados
- Insights sobre quais características influenciam a popularidade.
- Visualizações gráficas que mostram padrões interessantes no dataset.
- Experimentos com modelos preditivos.
